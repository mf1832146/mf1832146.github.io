To get machines to exhibit behaviour, which if done by humans, would be assumed to involve the use of intelligence. Samuel, 1983

Genetic Programming (GP)是进化算法(Evolutionary Computation, EC)的一种。和标准EC一样，GP需要维护族群(多目标)，更新族群（组合Crossover和变异Mutation）,过滤，最后实现解决方案。不同的是，GP维护的基本单元，或者说基本DNA,是代码。本章节主要介绍族群(population)的存储表示、初始化、更新和选择。
