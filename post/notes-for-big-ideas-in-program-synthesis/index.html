Notes for Big Ideas in Program Synthesis.

2023-11-28

  • Observational equivalence: Programs that behave the same on spec are redundant.

    (bottom-up enumerate. savings compound!)

    reference:

    • Recursive Program Synthesis (CAV 2013)
    • TRANSIT: specifying protocols with concolic snippets (PLDI 2013)

  • CEGIS: iteratively learn from counter-examples

    (inductive synthesizer + verification oracle) and can use constraint solver to implement both!

    reference:

    • Program sketching STTT'13 Armando Solar-Lezama.

  • Deductive synthesis: look for the proof the find the program. use the spec to guide the search.

    reference

    • Synthesis modulo recursive functions OOPSLA 2013
    • Structuring the Synthesis of Heap-Manipulating Programs POPL 2019
    • Type-and-example-directed program synthesis. PLDI 2015
